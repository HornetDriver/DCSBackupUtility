**********READ ME*************
This is a very simplified backup utility for the game DCS. 

1. You have the option to backup to two locations, if you only need to use one location just leave the 2nd location blank.

2. You can use the switch --silent to run the backup from the command line with the existing settings. You have to run it with the UI at least once first to set your defaults. This will allow you to call it from a scheduled task from windows.

3. As with all backups you need to test and verify your backups
